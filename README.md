# Authentication-app
This is just a simple Sign Up and Login application that loads a particular wallpaper if sign up and login details are valid.

# Stack
* Xml (frontend)
* Java (Backend)
* SqLite (Database)

# Features
No basic features included just a normal authentication-app.
* Updates coming soon.
